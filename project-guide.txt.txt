######## How to Setup and Run The  ######
    Downlode Project Source Code:https://github.com/ashish-code6/live-project/archive/refs/heads/main.zip
    Downlode Api Source Code:https://github.com/ashish-code6/live-project-api/archive/refs/heads/main.zip

Note : steps to setup the Project
    1. Download Source code from above url
    2. unzip/ extract both folder
    3. install node js 18+ version in your system
    4. install react devlopment enviroment using below command
        npm install -g create-react-app
    5. install JSON-SERVER to run api
        npm install -g json-server
    6. start the json-server using below command 
        a.open cmd in api folder (that u have extracted) and run
        
            json-server --watch data.json --port 1234
        
    7. start react devlopment server
        a. open cmd in react project folder that is "project1" and run this command
            npm start

        Note: You can browser https://localhost:3000
